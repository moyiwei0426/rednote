# Legacy Export Area

`github_data/` contains earlier GitHub-ready exports retained for compatibility.
It is not the primary discovery path for current device deliveries.

For device A and device C snapshots, begin at [`../data/README.md`](../data/README.md).
New device-level exports should use `data/<device>/<snapshot>/` and include a
snapshot-level `README.md` with scope, table definitions, counts, and privacy notes.

The existing `gaokao_2026_firstpass/` folder remains available as a historical export.
