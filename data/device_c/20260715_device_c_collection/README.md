## 字段说明

- `device_c_notes_unified.csv`：笔记 ID、标题、正文/描述、内容类型、发布时间（UTC 和上海时间）、作者标识、公开 IP 地域、互动数及来源文件。
- `device_c_comments_unified.csv`：笔记 ID、评论 ID、评论文本、评论时间（UTC 和上海时间）、评论者标识、评论者昵称、公开 IP 地域、点赞数及来源文件。
- `public_ip_location`：页面公开显示的 IP 属地；空值表示源页面未提供，不推断或补填。
- `creator_hash`：采集结果中的匿名化创作者标识，不等同于真实身份。
